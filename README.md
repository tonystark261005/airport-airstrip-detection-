# Airport Airstrip Detection System (YOLO + SAHI + GeoTIFF)

## Features
- Detects runways, hangars, airport structures
- Works on GeoTIFF satellite images
- SAHI-based tiled inference for large images
- Geospatial coordinate extraction

Offline deployment package available seperately.

## How to Run
```bash
pip install -r requirements.txt
python app.py
